# personal-book-manager
A personal movie library management application that allows users to store, rate, and organize their favorite movies while receiving suggestions based on their saved list.
A web application that helps users manage their personal movie library. 
Users can add movies with information such as title, genre, release year, and rating. 
The system allows users to select movie genres, rate movies after watching, view their saved movie list, and see the highest-rated movie. 
It also suggests movies based on the user's existing collection.
